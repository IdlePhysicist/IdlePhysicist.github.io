#2 - Git Tutorial / Commands

2018-02-09 21:57
#Git #Programming

Git is a program used for version control through github.com.

So here are the steps I use to create a repository:

git init
echo "# <Repo Name>" >> README.md
git add README.md
git commit -m "1st Commit"

Now I go to my github account page and hit “New Repository”, then type in a name and hit create. Back to the terminal:

git remote add origin <url to .git>
git push -u origin master

git add *
git commit -m “<message>”
git push

To save myself some time, I added the following line to my .bashrc file, or your .zshrc file if you are that way inclined.

gacp () { git add * && git commit -m "$1" && git push;}
